1.万物皆对象，其实不然。 在javascript中，一共又七种主要类型； String、Number、Boolean、Null、Undefined、Symbal、Object 前六种为基本类型，object为引用类型 typeof null // object 原理：不同的对象在底层都表示为二进制，在 JavaScript 中二进制前三位都为 0 的话会被判 断为 object 类型，null 的二进制表示是全 0，自然前三位也是 0，所以执行 typeof 时会返回“object”。 2.基本类型和引用类型。 基本类型和引用类型，归根结底要从两种类型各自特点出发。 基本类型包括上述6中。基本类型是一段简单的数据段，存在栈内存中。当复制基本类型时，其中一个改变，另一个不会变，比较时，按照值比较。 ----------- 引用类型：是一类对象所用的属性和方法。可能是由多个值构成的对象，存储在栈内容和堆内存中；当复制引用类型时，复制的时引用，一个改变，另一个也随之改变，比较时按其引用比较。 3.根据基本类型与引用类型各自的特点，便于区分。 4.使用instanceof来判断类型是否为Object时，引用类型会为true。基本类型为false
